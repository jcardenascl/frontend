> Frontend made with TailwindCSS for the Expense Tracker app

### 🏠 [Homepage](https://expensetrackerofficial.now.sh)

### ✨ [Demo](https://expensetrackerofficial.now.sh)

## Install

```sh
npm install
```

## Usage

```sh
npm run dev
```

## Run tests

```sh
npm test
```

in watch mode!

```sh
npm test -- --watch
```
